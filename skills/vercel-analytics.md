# Skill: Always Use Vercel Analytics

## When to Apply
When deploying any web application to Vercel.

## Guideline
• Always install and configure @vercel/analytics to track visitor stats
• **Next.js:** Install `@vercel/analytics`, add `<Analytics />` component to root layout
• **Other frameworks:** Use appropriate @vercel/analytics integration per framework docs

## Why
Provides free visitor analytics (pageviews, visitors, top pages, referrers, devices, geography) with zero configuration.
